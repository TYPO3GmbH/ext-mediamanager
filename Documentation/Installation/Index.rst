============
Installation
============

Install from git
^^^^^^^^^^^^^^^^

Alternatively, you can also get the latest development version from GitHub:


.. code-block:: bash

    $ git clone @todo


Install with composer
^^^^^^^^^^^^^^^^^^^^^

Install this TYPO3 Extension ``mediamanager`` via TYPO3 Extension Manager as usual, or via ``composer`` by
running:


.. code-block:: bash

    composer require typo3/cms-mediamanager



Head over to the Extension Manager module and activate the Extension.
